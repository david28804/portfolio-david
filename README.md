# Portfolio - KADEA

## Description
Ce projet est un portfolio personnel réalisé pendant la formation à KADEA.  
Il présente mon profil, mes projets et un formulaire de contact, avec une mise en page moderne en HTML/CSS.

---

## Technologies utilisées
- HTML5
- CSS3
- Git & GitHub

---

## Objectifs du projet
- Mettre en pratique l’intégration HTML5/CSS3 
- Reproduire une maquette en respectant les couleurs, les cartes et les boutons
- Organiser un projet clair (pages, styles séparés, images)
- Publier et versionner le code avec Git et GitHub
